This is an experimental open source system for integrating randomisation into AI roleplays. One system, yet it works for things like skill checks, plot-twists, combat, narrative pacing, solo roleplay oracles, etc. The idea is simple: Have the AI generate a weighted-probability roll table of 5 tag-based possibilities for what happens next, and have it roll 1d100 to pick one of these possibilities. In effect it functions as an on-the-fly rolltable for every response. Give the prompt a try with your favorite AI frontend. You will need a proper GM/narrator/character prompt in addition to the core Tablature prompt for the best experience. Simply append the Tablature prompt at the end of your roleplay prompt.  

I made this in very late 2025.
